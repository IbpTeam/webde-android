Mobile App UI framework

AppFramework UI is the User Interface/User Experience library for mobile applications. It uses HTML5 and CSS3 for animations and transitions. It provides javascript APIs to control the UI elements. It also provides vector icons and native looking UI themes for iOS, Android, Windows Phone and Blackberry.

Custom Side Menu
Side menu's are created using the <nav> tags. You can set the id and then refer to them on each panel by setting the data-nav='id' property. The side menu's will only show if you have <nav> defined.

Custom Footer
You should notice that the footer is no longer the default. You can create custom footers using the <footer> tag. You then set the id on each panel to change them using the data-footer='id' property.


Toggle Footer
You can toggle footer using $.ui.toggleNavMenu()

Custom Header
You should notice that the header is no longer the default. You can create custom header using the <header> tag. You then set the id on each panel to change them using the data-header='id' property.
